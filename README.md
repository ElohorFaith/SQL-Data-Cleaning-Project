Project Title: Data Cleaning with SQL

Overview: Cleaning and preparing raw data for SQL analysis is the main goal of this project. The objective was to find and fix common data problems in order to enhance data quality, consistency, and usability.

Key Tasks Performed: 
1. Remove Duplicates: To guarantee unique entries, duplicate records were removed.
2. Standardize the Data: Ensured data consistency was maintained hence we performed TRIM method, same spellings across the data, and formatted dates.
3. Null Values or Blank Values: Updated Null values column 
4. Remove any columns or rows: Removed irrelevant columns and rows in the dataset.

Tools: MYSQL, MYSQL Workbench

Outcome: a clean, consistent dataset that is prepared for viewing and analysis. This procedure improved the overall efficiency of ensuing queries and increased data dependability.
